One biggest issue that I had was the HTML sytax issue. My natural intuition is to keep writing code and paragraph as long as the content is going to continue downward, but the output usually looks very messy or not showing up as what I wanted. Therefore, I have later learned more tags within html, such as "li" and "body" tags, and how to wrap paragraph and image inside one "div" so that they appear grouped together.

CSS Transition

I have embedded three CSS transition. The first one is the background color change on the home page when the user hover their mouse into the webpage. This is a welcoming effect and separates out the the navigation bar and the background contents to give a fresh impression to the user. The other two are just the hover effect when the user hover their mouse onto an item to give the indication that the link/picture is clickable. Specifically, the user can hover over the option on the navigation bar and the background color of the option will change. Another one is when the user hover over the picture to a project or the WP0 writeup, the picture background brightness will change to show the clickable indication. I have made these transitions by adding the .hover and .focus aspects into the CSS sheets and specified the transition time that I desire to happen.
