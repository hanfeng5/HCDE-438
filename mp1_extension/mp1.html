Browser Extension Writeup 🖥

Overview and Usage

The core functionality of my browser extension is to help the users to make a decision when they come across different options as they browse through the websites. For example, this extension would be helful when the users are debating which color to choose when they are online shopping, if they should go to their favorite restaurant tonight instead of the weekend, and so much more.

This extension is being stored on the GitHub repository browser-extensions . Inside the repository, there is the folder "decision-maker", which could be loaded directly into Google Chrome as unpacked extension.

Issue Deep-Dive

The biggest issue that I ran into was the part that no matter which option does the user choose to make the decision, both decision mechanisms will run one after another. In another word, there is no outcome difference when the user choose one decision mechanism over another. This is definitely a logic error because I have the correct syntax, but I had a hard time to figure out where did my logic gone wrong. The way I solved was by asking for my classmate to look over my code and give me some debugging hints. Turns out that was very effective that I was overly relying on the previous constructed code for the blocker extension and did not realize that now I have two options to choose from in comparsion to one in the past. Therefore, I have implemented the code to identify which button did the user click and which specific decision making mechanism to use.

Ideas and Future Work

Looking ahead, I wished that I could have elaborate the "option-based" decision making mechanism a bit more elaborated. For example, I was imagining to ask the user to input their specific options and store these information as a list. When the system makes the decision, it will directly tell the user about the option details as they have entered instead of stating the option number. In order to do this, I need a good user interface as well that has a next button, a cancel button, and a done button. It is definitely what I will explore next before I officially submit this extension onto the Chrome extension store.
