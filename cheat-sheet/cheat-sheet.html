CSS Styles:

  • justify-content: center -- This will align the contents, such as a group of pictures, to the middle of the page, or the container if specified.
  • nav a {transition: all 0.5s ease-out;} + nav a:hover, nav a:focus {background-color: #FA9500;color: #333;} -- First, this command will only apply to the anchor elements within the nav property. Then, it is saying when the user hover the mouse, or focus on the element, the CSS styles will be having "these" colors. But first, there need to be a "transition" declaration within the nav a styles to indicate the desired effect in changing between two states when the user trigger this CSS transition.
  • font-family: 'Fredoka One', cursive; -- This specifies the font that will be used within the corresponding container. The ones with "" is usually the special font that may require additional installation. It is usually followed by a "basic" font to tell the website to dispaly this alternative solution if the previous desired font is not available on user's side.
