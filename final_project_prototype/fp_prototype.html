Overview and Usage

This to-do planner is a implementation of the paper planner/to-do list but the user can simply keep track of their daily commitments on through this website. This to-do planner is built with "React" framework and currently achieve the basic functionality. The users can add new to-do items in the input box which will be queued at the end of the list, and the user can delete the task or toggle the checkbox to mark the task as done. Based on the amount of the remaining tasks, the website will remind users how many remaining tasks are left inside the to-do list.

To experience this virtual to-do planner, you may access it from To-Do-Planner Prototype

Issue Deep-Dive

One issue happened when I was attemtping to use the "useState" function within React to change the state of the variable and pass it into a function to change the original value of the variable. When I was reading about this function, I wanted to pair it with another "function" so that the program will add the task when the user hit the "add" button on the to-do list. However, when I wrote out the function, I have been constantly getting the error of scope not found. I have gotten some help from Hannah and understand the "class" and "function" logic within Javascript. I was very used to other coding languages where a function, or a method, is exclusive to itslef with the return statement to end the method. However, instead of writing the component files with the "class" keyword and extend the React library, we can write the object as a large function. Then, there can be sub functions inside so that other variables can access to the scope.
