P5 Sketches with Creative Code

Overview and Usage

With this porject, I have made creative code in the p5.js library to generate different graph patterns and styles within the topic of "heart". Each individual graphs has their own uniqueness based on the title, just like the "sound heart" will change sizes based on the music rythms, and "interactive heart" will change size based on the user's interaction. Everything is linked to the P5 Sketch Gallery where you can click into the individual projects and have an immersive experience with these creative designs.

Issue Deep-Dive

One of the issue I have ran into was to successfully load the mp3 file into my "sound heart" code to make cool effect when the music is playing. I have tried to follow the steps and tutorials on the P5 example websites, but all the example code do not load my mp3 file correctly and my code will always stuck at the loading screen without drawing anything at all. Turns out, the keyword "assets" is not needed on line 4 when I am loading the mp3 file, since the mp3 file lives under the same directory of my sketch.js code. By removing the assets keyword, my code runs perfectly and the music will play after the mouse is clicked.
